Allowance Allocator

GitHub

I built this application in order to originally serve as a medium to compensate and incentivize my little brother to get into coding.

Users can login to complete tasks or create and modify tasks that are tied to the allowance receiver's account.

Happy Hour Las Vegas

GitHub

I built this application as a repository for restaurants and bars in the Las Vegas area that offer Happy Hour specials.

The public view can see all locations I've currently added while account holders can log in and view the details of the specials as well as post comments.

Noteful

GitHub

This was my first full stack app that was built whilst attending my bootcamp program.

It is a note organization application that allows anyone to add notes that are designated to a specific folder and sort by folder.
